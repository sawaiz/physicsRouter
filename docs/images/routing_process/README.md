## Regenerate

```bash
# From repo root (venv + native C++ module built)
bash scripts/build_native.sh
PYTHONPATH=native/build:src python scripts/render_routing_process.py --halo
```

Metadata: `render_meta.json` (both boards, timings, route status, DRC),
`drc_report.json` (canonical HALO-90 report).

The renderer writes stage 7 and `drc_report.json` directly. Copper-area fill
and thermals are deferred to KiCad; the native report validates area
boundaries plus committed tracks and vias.
